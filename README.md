# YouTube Monitor Bot

Telegram-бот для мониторинга YouTube каналов и поиска видео.

## Функции

- 🔍 **Поиск видео** — `/search <запрос> <дни>` — поиск через yt-dlp (без квот)
- 📺 **Мониторинг каналов** — `/add_channel <URL>` — добавление канала
- 📋 **Список каналов** — `/list_channels` — просмотр подписок
- 🔄 **Проверка сейчас** — `/check_now` — принудительная проверка
- ⭐ **Последние видео** — `/recent` — последние из подписок

## Установка

```bash
git clone https://github.com/LeontyV/youtube-monitor-bot.git
cd youtube-monitor-bot
pip install -r requirements.txt
cp .env.example .env
# Заполните .env
python bot.py
```

## Автоматическая проверка каналов

Проверка запускается автоматически каждые 30 минут (cron).

**Защита от превышения квоты:**
- Если API квота исчерпана — проверки автоматически отключаются
- Возобновляются в полночь UTC следующего дня
- Бот отправит уведомление когда квота закончится

## Команды

| Команда | Описание |
|---------|----------|
| `/search <слова> <дни>` | Поиск (yt-dlp, без квот) |
| `/add_channel <URL>` | Добавить канал |
| `/list_channels` | Список каналов |
| `/check_now` | Проверить сейчас |

## Технологии

- Python 3 + python-telegram-bot
- yt-dlp (поиск без квот)
- YouTube Data API v3 (мониторинг каналов)
- SQLite
