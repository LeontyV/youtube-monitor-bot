# YouTube Monitor Bot

Telegram-бот для мониторинга YouTube каналов и поиска видео.

## Функции

- 🔍 **Поиск видео** — `/search <запрос> <дни>` — поиск через yt-dlp (без квот)
- 📺 **Мониторинг каналов** — `/add_channel <URL>` — добавление канала
- 📋 **Список каналов** — `/list_channels` — просмотр подписок
- 🔄 **Проверка сейчас** — `/check_now` — принудительная проверка
- ⭐ **Последние видео** — `/recent` — последние из подписок
- 📡 **Прямые эфиры** — автоматически детектит `/live` редиректы

## Установка

```bash
git clone https://github.com/LeontyV/youtube-monitor-bot.git
cd youtube-monitor-bot
pip install -r requirements.txt
cp .env.example .env
# Заполните .env
python bot.py
```

## Автоматическая проверка каналов

Проверка запускается автоматически каждые 10 минут.
- **Параллелизм** — до 3 каналов одновременно (через semaphore)
- **Быстро** — yt-dlp парсит только первые 5 видео (`playlist_items`)
- **Live** — быстрая проверка `/live` страницы с обработкой редиректа

## Команды

| Команда | Описание |
|---------|----------|
| `/search <слова> <дни>` | Поиск через yt-dlp |
| `/add_channel <URL>` | Добавить канал |
| `/list_channels` | Список каналов |
| `/check_now` | Проверить сейчас |
| `/recent` | Последние видео |
| `/status` | Статус мониторинга |
| `/filters` | Фильтры |

## Технологии

- Python 3 + python-telegram-bot
- yt-dlp (мониторинг и поиск без API ключа)
- SQLite
- WireGuard VPN (для доступа к YouTube из РФ)
